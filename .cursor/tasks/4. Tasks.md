## Task 1: Create Interview Template Handler

**Status**: ✅ Complete

**Description**: Create a new handler file for interview template (interview plan) operations.

**Files to Create**:
- `backend/internal/api/handlers/interview_templates.go`

**Steps**:
1. Create `InterviewTemplateHandler` struct with `q *db.Queries` field
2. Create `NewInterviewTemplateHandler(q *db.Queries)` constructor
3. Define `CreateInterviewTemplateRequest` struct:
   - `RubricID` (uuid.UUID, required, json tag: "rubricId")
   - `Title` (string, required, json tag: "title")
   - `Instructions` (string, optional, json tag: "instructions")
   - `Config` (json.RawMessage, optional, json tag: "config")
   - `Status` (string, optional, json tag: "status")
   - `CurriculumSubject` (string, optional, json tag: "curriculumSubject")
   - `CurriculumLevelBand` (string, optional, json tag: "curriculumLevelBand")
4. Define `InterviewTemplateResponse` struct matching `AppInterviewPlan` fields:
   - `InterviewPlanID` (uuid.UUID, json tag: "interviewPlanId")
   - `RubricID` (uuid.UUID, json tag: "rubricId")
   - `Title` (string)
   - `Instructions` (string, nullable)
   - `Config` (json.RawMessage or map[string]interface{})
   - `Status` (string)
   - `CurriculumSubject` (string, nullable)
   - `CurriculumLevelBand` (string, nullable)
   - `CreatedAt` (pgtype.Timestamptz)
   - `UpdatedAt` (pgtype.Timestamptz)
5. Implement `CreateInterviewTemplate` method:
   - Decode JSON request body using `json.NewDecoder(r.Body).Decode(&req)`
   - Validate required fields:
     - `rubricId` must not be `uuid.Nil`
     - `title` must not be empty (after trimming)
   - Validate `status` enum (if provided):
     - Must be one of: "draft", "in_progress", "completed"
     - If not provided, default to "draft"
   - Verify rubric exists:
     - Call `h.q.GetRubricByID(ctx, rubricID)` with `pgtype.UUID`
     - If returns `pgx.ErrNoRows`, return 404 Not Found
     - If other error, return 500 Internal Server Error
   - Handle `config` JSON:
     - If `config` is `nil` or empty, default to `{}`
     - Convert `json.RawMessage` to `pgtype.JSONB` for database
   - Convert request to `CreateInterviewPlanParams`:
     - `rubricId`: `uuid.UUID` → `pgtype.UUID`
     - `title`: `string` (required)
     - `instructions`: `string` → `pgtype.Text` (nullable)
     - `config`: `json.RawMessage` → `pgtype.JSONB` (default to `{}`)
     - `status`: `string` → `string` (enum, default "draft")
     - `curriculumSubject`: `string` → `pgtype.Text` (nullable)
     - `curriculumLevelBand`: `string` → `pgtype.Text` (nullable)
   - Call `h.q.CreateInterviewPlan(ctx, params)`
   - Convert response from `AppInterviewPlan` to `InterviewTemplateResponse`:
     - `pgtype.UUID` → `uuid.UUID` (check `.Valid`)
     - `pgtype.Text` → `string` (use `.String` if `.Valid`, else empty string or null)
     - `pgtype.JSONB` → `json.RawMessage` or keep as JSONB
     - `pgtype.Timestamptz` → keep as-is (serializes to JSON)
   - Return 201 Created with response
   - Set `Content-Type: application/json` header
   - Handle errors:
     - 400 for validation errors (missing fields, invalid formats, invalid enums)
     - 404 for rubric not found
     - 500 for database errors

**Request/Response**:
- Request: POST with JSON body (see PRD section 4.2)
- Response: 201 Created with `InterviewTemplateResponse`
- Errors: 400 (validation), 404 (rubric not found), 500 (DB error)

**Acceptance Criteria**:
- Validates all required fields (`rubricId`, `title`)
- Verifies rubric exists before creating plan
- Defaults `status` to "draft" if not provided
- Defaults `config` to `{}` if not provided or null
- Validates `status` enum values
- Properly converts between `pgtype` and standard types
- Returns appropriate HTTP status codes
- Handles nullable fields correctly (`instructions`, `curriculumSubject`, `curriculumLevelBand`)

## Notes

- Follow the pattern established in `handlers/rubrics.go`
- Use `pgtype` types for database operations and convert to standard types for JSON responses
- Validate UUIDs using `github.com/google/uuid` package
- Handle JSON marshaling/unmarshaling carefully for `config` JSONB field:
  - `json.RawMessage` is good for preserving JSON structure
  - Can also use `map[string]interface{}` or `pgtype.JSONB` directly
- Default values should be applied in handler (not just relying on database defaults)
- Verify rubric exists before creating plan to provide better error messages (404 vs 500)
- Handle `pgx.ErrNoRows` specifically for "not found" cases
- Nullable fields should return `null` in JSON (not empty string) when not set
- Error messages should be user-friendly but not expose internal details
- Consider using a helper function for UUID validation if used in multiple handlers

Based on existing `CreateRubric` handler pattern:

```go
type InterviewTemplateHandler struct {
    q *db.Queries
}

func NewInterviewTemplateHandler(q *db.Queries) *InterviewTemplateHandler {
    return &InterviewTemplateHandler{q: q}
}

func (h *InterviewTemplateHandler) CreateInterviewTemplate(w http.ResponseWriter, r *http.Request) {
    var req CreateInterviewTemplateRequest
    if err := json.NewDecoder(r.Body).Decode(&req); err != nil {
        http.Error(w, "invalid JSON body", http.StatusBadRequest)
        return
    }
    
    // Validate required fields
    if req.Title == "" {
        http.Error(w, "title is required", http.StatusBadRequest)
        return
    }
    if req.RubricID == uuid.Nil {
        http.Error(w, "rubricId is required", http.StatusBadRequest)
        return
    }
    
    // Validate status enum
    status := req.Status
    if status == "" {
        status = "draft" // default
    } else if status != "draft" && status != "in_progress" && status != "completed" {
        http.Error(w, "invalid status: must be one of draft, in_progress, completed", http.StatusBadRequest)
        return
    }
    
    ctx := r.Context()
    
    // Verify rubric exists
    rubricIDPgtype := pgtype.UUID{Bytes: req.RubricID, Valid: true}
    _, err := h.q.GetRubricByID(ctx, rubricIDPgtype)
    if err != nil {
        if errors.Is(err, pgx.ErrNoRows) {
            http.Error(w, "rubric not found", http.StatusNotFound)
            return
        }
        http.Error(w, "failed to verify rubric", http.StatusInternalServerError)
        return
    }
    
    // Handle config default
    configJSONB := pgtype.JSONB{}
    if len(req.Config) == 0 {
        configJSONB.Bytes = []byte("{}")
        configJSONB.Valid = true
    } else {
        // Parse and validate JSON
        configJSONB.Bytes = req.Config
        configJSONB.Valid = true
    }
    
    // Convert to database params and create...
}
```
