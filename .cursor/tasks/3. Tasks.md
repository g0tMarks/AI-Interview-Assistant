## Task 1: Implement GET /rubrics Handler

**Status**: ã Complete

**Description**: Add handler method to list all rubrics for a teacher.

**Files to Modify**:
- `backend/internal/api/handlers/rubrics.go`

**Steps**:
1. Add `ListRubrics` method to existing `RubricHandler` struct
2. Extract `teacherId` from query parameter using `r.URL.Query().Get("teacherId")`
3. Validate `teacherId` is present (not empty)
4. Validate `teacherId` is a valid UUID using `github.com/google/uuid`
5. Convert `teacherId` string to `uuid.UUID`
6. Convert `uuid.UUID` to `pgtype.UUID` for database query:
   ```go
   teacherID := pgtype.UUID{
       Bytes: parsedUUID,
       Valid: true,
   }
   ```
7. Call `h.q.ListRubricsByTeacher(ctx, teacherID)`
8. Handle database errors:
   - If `pgx.ErrNoRows`, return empty array `[]` (not an error)
   - If other error, return 500 Internal Server Error
9. Convert `[]AppRubric` to `[]RubricResponse`:
   - Loop through results
   - Convert each `AppRubric` to `RubricResponse`:
     - `pgtype.UUID` ã `uuid.UUID` (check `.Valid` before conversion)
     - `pgtype.Text` ã `string` (use `.String` if `.Valid`, else empty string)
     - Keep `pgtype.Timestamptz` as-is (serializes to JSON correctly)
10. Return 200 OK with JSON array
11. Set `Content-Type: application/json` header

**Request/Response**:
- Query param: `teacherId` (required, UUID)
- Response: 200 OK with array of `RubricResponse`
- Errors: 400 (invalid/missing teacherId), 500 (DB error)

**Acceptance Criteria**:
- Returns empty array `[]` if teacher has no rubrics (not an error)
- Returns 400 if `teacherId` is missing
- Returns 400 if `teacherId` is invalid UUID format
- Returns rubrics ordered by `created_at DESC` (handled by SQL query)
- All fields properly converted from `pgtype` to standard types
- Returns 200 OK with array (even if empty)
- Handles null `description` field correctly (returns empty string or null in JSON)

## Notes

- Handler method should be added to existing `RubricHandler` struct (no new handler needed)
- Reuse existing `RubricResponse` struct for consistency with POST /rubrics
- Follow the same error handling pattern as `CreateRubric` method
- Use `github.com/google/uuid` package for UUID validation and parsing
- Handle `pgtype` types carefully:
  - Check `.Valid` before accessing `.Bytes` for UUIDs
  - Check `.Valid` before accessing `.String` for Text fields
  - `pgtype.Timestamptz` can be serialized directly to JSON
- Empty array response is a success case (200 OK), not an error
- Query parameter extraction: `r.URL.Query().Get("teacherId")`
- Consider trimming whitespace from query parameter before validation
- Error messages should be user-friendly but not expose internal details

```go
func (h *RubricHandler) ListRubrics(w http.ResponseWriter, r *http.Request) {
    // Extract and validate teacherId query parameter
    teacherIdStr := r.URL.Query().Get("teacherId")
    if teacherIdStr == "" {
        http.Error(w, "teacherId is required", http.StatusBadRequest)
        return
    }
    
    teacherID, err := uuid.Parse(teacherIdStr)
    if err != nil {
        http.Error(w, "invalid teacherId format", http.StatusBadRequest)
        return
    }
    
    ctx := r.Context()
    
    // Convert to pgtype.UUID
    teacherIDPgtype := pgtype.UUID{
        Bytes: teacherID,
        Valid: true,
    }
    
    // Query database
    rubrics, err := h.q.ListRubricsByTeacher(ctx, teacherIDPgtype)
    if err != nil {
        // Handle errors appropriately
        http.Error(w, "failed to retrieve rubrics", http.StatusInternalServerError)
        return
    }
    
    // Convert to response format
    resp := make([]RubricResponse, len(rubrics))
    for i, rubric := range rubrics {
        // Convert each rubric...
    }
    
    w.Header().Set("Content-Type", "application/json")
    w.WriteHeader(http.StatusOK)
    json.NewEncoder(w).Encode(resp)
}
```
