## Notes

- **Database Connection**: Decide whether to use a real test database or mock/in-memory database. For integration tests, a real database is typically preferred.
- **Test Isolation**: Consider using database transactions that rollback after each test, or use a separate test database that gets cleaned up.
- **Message Creation**: If there's no HTTP handler for creating messages, use the database queries directly. This is acceptable for integration tests.
- **Handler vs HTTP**: Decide whether to call handlers directly or make HTTP requests. Direct handler calls are faster, HTTP requests test the full stack.
- **Test Data**: Use realistic but clearly test data (e.g., "Test Rubric", "Test Template") to distinguish from production data.
- **UUIDs**: All IDs should be validated as proper UUIDs.
- **Timestamps**: Verify that `createdAt` and `startedAt` timestamps are set and reasonable.
- **Relationships**: Verify all foreign key relationships are correct (interview → template → rubric → teacher).
- **Nullable Fields**: Handle nullable fields correctly (teacherId, studentName, completedAt, interviewQuestionId in messages).

## Code Structure Example

```go
func TestCreateRubricTemplateInterviewFlow(t *testing.T) {
    // Setup
    ctx := context.Background()
    db := setupTestDB(t)
    defer teardownTestDB(t, db)
    
    queries := db.NewQueries(db)
    teacher := createTestTeacher(t, queries, ctx)
    
    // Task 2: Create Rubric
    rubric := createTestRubric(t, queries, ctx, teacher.TeacherID)
    
    // Task 3: Create Interview Template
    template := createTestInterviewTemplate(t, queries, ctx, rubric.RubricID)
    
    // Task 4: Create Interview
    interview := createTestInterview(t, queries, ctx, template.InterviewPlanID, teacher.TeacherID)
    
    // Task 5: Add Messages
    message1 := createTestMessage(t, queries, ctx, interview.InterviewID, "ai", "Hello!")
    message2 := createTestMessage(t, queries, ctx, interview.InterviewID, "user", "Hi there!")
    
    // Task 6: Retrieve and Assert
    retrievedInterview := getInterview(t, queries, ctx, interview.InterviewID)
    assertInterviewStructure(t, retrievedInterview, interview, template, rubric)
    
    messages := getInterviewMessages(t, queries, ctx, interview.InterviewID)
    assertMessages(t, messages, []AppInterviewMessage{message1, message2})
}
```
